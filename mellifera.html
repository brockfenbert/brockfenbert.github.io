Mellifera

I've always been interested in game design and think I have some good ideas. I took a Game Programming class through Northeastern in Summer 2020 to get more practice working on games. Mellifera was created in C# and Unity over 6 weeks and a team of 4. I had a lot of fun working on this product and think the end-result turned out pretty well (although we have found a few bugs since release). Feel free to give the game a try or checkout the game trailer. GitHub and source code available upon request.
